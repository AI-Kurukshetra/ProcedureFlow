# Contributing

## Scope
This project targets a production-ready Next.js UI. Follow the UI rules and skill guidelines in `RULES.md` and `SKILL.md`.

## Workflow
1. Create a focused branch.
2. Keep changes small and cohesive.
3. Run `npm run build` before PR.

## Code Standards
- Preserve server/client boundaries.
- Avoid unnecessary dependencies.
- Prefer reusable UI components.
- Keep styles consistent and minimal.

## PR Checklist
- No breaking changes to routes or SEO.
- Responsive on mobile/tablet/desktop.
- Loading/empty/error states handled.
- Accessibility verified.
